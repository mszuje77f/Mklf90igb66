### heroku上部署v2ray

本项目只支持部署vless+ws+tls 或 vmess+ws+tls单协议节点，支持跳转伪装网页，伪装网页固定为3D元素周期表。

path路径自动设置为：/自定义UUID码-vless 或 /自定义UUID码-vmess
